# bt-shop
### Ecommerce website
### Technologies used:

# Installation
- git clone https://github.com/binzam/bt-shop_ecommerce-website.git
- cd bt-shop_ecommerce-website/backend
- npm install
- cd ../frontend
- npm install
- Open browser
- Open link http://localhost:5555
